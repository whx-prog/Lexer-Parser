# Lexer-Parser
语法分析表达式生成树
将输入字符流进行词法分析后得到Token流,并对不同类型的词进行染色
对Token流进行语法判断
已包含的表达式终结符包括:+ - * / % () [] == != < > <= >= sizeof()
